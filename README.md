# 🃏 Phantom Chat

A real-time chat application with a Persona 5 inspired UI.

## 🚀 Live Demo

[persona5-chat.vercel.app](https://persona5-chat.vercel.app)

## ✨ Features

- Google Sign In + Email/Password auth
- Public chat rooms (create your own)
- Direct messages between users
- Persona 5 styled UI with theme switcher
- Real-time messaging via Firebase

## 🛠️ Built With

- React + Vite
- Firebase (Auth + Firestore)
- React Router DOM

## 🔧 Run Locally

git clone https://github.com/016HaiderAli/persona5-chat.git
cd persona5-chat
npm install

# Add your .env file with Firebase keys

npm run dev
